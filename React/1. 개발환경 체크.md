### 1.1.1. Node.js

#### 1) LTS

- 더 안정화된 버전

#### 2) Current

- 최신의 버전 - LTS보다는 조금 불안정하다.

<br/>

#### 3) NVM 설치 이후

- 특정 node 버전 설치

```shell
nvm install 14.16.1  
```

- 사용

```shell
nvm use 14.16.1
```

- 기본 버전 지정

```shell
nvm alias default 14.16.1
```
